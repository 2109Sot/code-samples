# technical_memo
This repository contains selected methodologies, code notebooks and workflow documentation from investigative reporting projects I have worked. Across these investigations, I built technical workflows to retrieve, structure and analyze complex public records in order to support accountability reporting. The projects span multiple domains — maritime trade, court archives, distressed property markets and election administration — but share a common methodological challenge: crucial public-interest data exists, yet it is fragmented, inaccessible, or poorly structured. The code and documentation here illustrate how these reporting systems were developed. Some datasets and scripts have been simplified or partially redacted to respect legal, ethical, or editorial constraints.
